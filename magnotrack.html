How to Run on Your Mobile Phone

1
HTTPS is Mandatory

Mobile browsers enforce strict sandboxes. The Web Magnetometer and Vibration APIs only operate under secure origins (https://) or localhost.

2
Enable Sensor Flags (Chrome/Android)

In Chrome on Android, open a tab to: chrome://flags/#enable-generic-sensor-extra-classes and set it to Enabled, then relaunch Chrome.

3
Add to Home Screen (PWA Mode)

Tap your browser's options menu () and select "Add to Home Screen". This launches MagnoTrack in standalone fullscreen mode without distracting browser bars.

Web API Implementation Code

This single-file app is constructed with the standard Web Generic Sensor standard. Here is the exact JS architecture required to connect hardware magnetometer changes directly with device vibration: 

// 1. Initializing the Web Magnetometer API
if ('Magnetometer' in window) {
  const sensor = new Magnetometer({ frequency: 60 });
  
  sensor.addEventListener('reading', () => {
    // Total scalar field length calculations
    const mag = Math.sqrt(
      sensor.x * sensor.x + 
      sensor.y * sensor.y + 
      sensor.z * sensor.z
    );

    // 2. Triggering Physical Haptic Buzz
    if (mag > threshold && 'vibrate' in navigator) {
      // Short dynamic feedback pulse
      navigator.vibrate(100); 
    }
  });

  sensor.start();
}
Note on iOS Support: Apple Safari currently blocks standard Web Generic Sensors due to privacy rules regarding browser tracking vector fingerprints. iOS users can enjoy the high-performance desktop simulator!

Web Deployment Guide

To run this tool on your physical device, host this index.html on any free secure web space (such as GitHub Pages, Vercel, or Netlify).

Testing without deploying (Local Dev):

  1. Run a local dev server on your computer (e.g., using npx serve or Live Server extension).
  2. Find your computer's local IP address (e.g., 192.168.1.X).
  3. Open that address in Chrome on your mobile phone. (Chrome allows local network HTTP addresses to use Web APIs as secure origins).
Pro-Tip: Once loaded on your phone browser, enable Chrome's "Install" feature to run MagnoTrack like a native system application!
